# R74 — Build-To-Learn Pathways

Status: Drafted for implementation
Requirement: R74
Task: Task 168

## Purpose

Define the first bounded build-to-learn pathway so the learning layer can turn a concept into an intentional experiment rather than stopping at explanation or note capture.

## Core behavior

The operator should be able to:
- choose a concept to learn by building
- receive a bounded experiment suggestion
- understand what project or workflow the experiment anchors to
- save the pathway into the private learning layer for later execution and reflection

## First-slice boundaries

The first slice should:
- stay private-first and local-only
- produce a bounded experiment path, not a full project plan
- connect the concept to current OS work where possible
- preserve a clear success signal and capture prompt for later learning review

The first slice should not:
- auto-create a full project
- auto-implement the experiment
- sprawl into broad research planning

## Done definition

Task 168 should be considered complete when:
- the OS can turn at least one concept into a bounded build-to-learn pathway
- the pathway is saved into the private learning layer
- the result feels like an intentional learning experiment rather than a vague aspiration
