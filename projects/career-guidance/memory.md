### O1 — Multiple-posting input needs explicit separation guidance

Observation:
- The target job descriptions field needs concrete guidance for users pasting multiple postings.

Evidence:
- R2 reports user uncertainty about acceptable multiple job posting input methods.
- The pre-R2 UI said users could paste one or more descriptions but did not show how to separate postings.

Confidence:
- High

Validation method:
- Experience Designer review of R2 and the current Streamlit form copy.

Implication:
- Future input-flow changes should provide concrete examples at the point of entry when a field accepts multiple items.
