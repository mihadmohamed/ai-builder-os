# Wright Sparks website showcase

AI Builder OS was used to turn an existing electrical-services website into a responsive, production-ready web experience for Wright Sparks Ltd.

## Delivered outcome

- Recreated the client’s established brand and service content in a modern Next.js site.
- Added responsive navigation, service and accreditation sections, customer-review evidence, and clear quote/contact actions.
- Added the Checkatrade trust widget requested after the initial release.
- Verified desktop and mobile rendering, navigation targets, and horizontal overflow before release.
- Published the production website at [ai-builder-os-hazel.vercel.app](https://ai-builder-os-hazel.vercel.app).

The live project is governed by AI Builder OS from a separate client-controlled repository. Its source code, canonical product history, workspace location, and repository metadata are intentionally not part of this public showcase.
