# Web App Frontend Capability Bundle

This is the local AI Builder OS frontend capability bundle for `web_app` projects.

It is intentionally bounded to the first slice of frontend support:

- frontend app builder
- frontend testing and debugging
- React best practices
- shadcn/ui best practices

This bundle does not include:

- Stripe or payment workflows
- database modeling or storage architecture
- auth provider implementation selection
- broad backend or infrastructure capability

Use this bundle when the project or requirement runtime is explicitly `web_app`.
